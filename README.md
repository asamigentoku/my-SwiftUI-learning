# my-SwiftUI-learning 🚀

SwiftUIの学習記録とサンプルコードを管理するリポジトリです。

## 📌 学習の目標
- SwiftUIの基本コンポーネント（View）をマスターする
- 状態管理（@State, @Bindingなど）を理解する
- 1つ以上のオリジナルミニアプリを完成させてリリースする

## 📚 参考リソース（公式）
- [Introducing SwiftUI Tutorials](https://developer.apple.com/tutorials/swiftui/)
- [SwiftUI Documentation](https://developer.apple.com/documentation/swiftui/)

## 🛠️ 学習ロードマップ & 進捗
- [ ] Day 1: 基礎レイアウト（`VStack`, `HStack`, `ZStack`）
- [ ] Day 2: 状態管理（`@State`, `@Binding`）
- [ ] Day 3: リストとナビゲーション（`List`, `NavigationStack`）
- [ ] Day 4: アニメーションとインタラクション
- [ ] Day 5: データの永続化（SwiftData / CoreData）

## 📝 開発環境
- macOS 14+ (Sonoma)
- Xcode 15+
- Swift 5.9+ / SwiftUI
